# bootcamp-bradesco-genai-dados-cyber

Bem-vindo ao meu repositório oficial do **Bootcamp Bradesco - Inteligência Artificial, Dados e Cibersegurança**, oferecido pela [DIO.ME](https://web.dio.me). Este espaço reúne todas as minhas entregas, anotações e projetos práticos desenvolvidos ao longo da jornada.

## 📌 Sobre o Bootcamp
Aprenda do zero como utilizar **Inteligência Artificial Generativa**, **Análise de Dados** e **Cibersegurança** para construir soluções modernas e alinhadas às principais demandas do mercado.

**Principais tópicos abordados:**
- 🤖 IA Generativa e Prompt Engineering
- 📊 Análise de dados com Excel, SQL e Python
- 🔒 Práticas essenciais de segurança digital e automação
- 🛠️ Desenvolvimento de assistentes inteligentes
- 📈 Tratamento e visualização de dados
